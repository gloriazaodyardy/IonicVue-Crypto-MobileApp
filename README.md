## Run Locally

**Clone the project**

```

git clone https://github.com/gloriazaodyardy/IonicVue-Crypto-MobileApp.git

```

**Go to the project directory**

```

  cd IonicVue-Crypto-MobileApp

```

**Install dependencies**

```

  npm install

```

**Start the server**

```

  ionic serve

```
> **Note:** Make sure Ionic CLI is installed globally: `npm install -g @ionic/cli`
